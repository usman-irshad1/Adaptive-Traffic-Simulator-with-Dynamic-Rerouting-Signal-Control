{\bfseries{Adaptive Traffic Flow Optimization System (C++)}}

An Intelligent Transportation System (ITS) simulation that models and optimizes urban traffic flow using graph algorithms, congestion modeling, and adaptive signal control.

This project simulates real-\/world traffic conditions in major Pakistani cities such as Karachi, Lahore, Multan, and Peshawar, with the goal of reducing congestion and improving travel efficiency.

{\bfseries{Core Features}}

{\bfseries{1. Dynamic Graph-\/\+Based Road Network}}


\begin{DoxyItemize}
\item Cities and intersections are modeled as nodes
\item Roads are modeled as directed edges
\item Enables realistic simulation of traffic flow between locations
\end{DoxyItemize}

{\bfseries{2. Intelligent Routing (Dijkstra Algorithm)}}


\begin{DoxyItemize}
\item Computes shortest paths using dynamic edge weights
\item Edge weights update in real time based on congestion
\item Vehicles automatically reroute based on current traffic conditions
\end{DoxyItemize}

{\bfseries{4. Adaptive Traffic Signals (Longest Queue First)}}


\begin{DoxyItemize}
\item Intersections monitor incoming road queues
\item The road with the highest queue length is given the green signal
\item Helps reduce bottlenecks and improves flow at congested intersections
\end{DoxyItemize}

{\bfseries{5. Backpressure Flow Control}}


\begin{DoxyItemize}
\item Enforces road capacity constraints
\item Vehicles cannot enter a road if it is full
\item Prevents unrealistic movement and models real traffic jams
\end{DoxyItemize}
